Download Multiple Attachments

This module will allow users to Download Multiple Attachments. Select Attachments and click on an Action button on top, It has a sub-button called Download.

Added a separate menu with a name Attachment, You can Search for the document and apply filters according to your need. Create a Zip file of the selected records and download it.

Features

  • This module is user-friendly.
  • This module reduces human effort.
  • Added separate menu with name Attachment in Settings -> Attachments.
  • Go to List view and select your attachments records which you want to download as Zip.
  • One Click Download from the Action button.
  • Selected records will automatically added in a new Zip.
  • You can Filter and Search the attachment records as per your need.
  • Useful when you want to download multiple files from the attachment in one go.
