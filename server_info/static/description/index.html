Server Info

Settings tab with automatically updated server informations

Version: V17.0.1.1.0

Created and maintained by
Mythrive.pl
assistance and support
kontakt@mythrive.pl

Description

This module add new tab 'Server Info' in settings. This tab contains automatically updated informations about server cpu, ram and disk usage. It also allows you to change frequency of informations updates.

Usage

See your Server Informations

- Go to Settings
- Select 'Server Info' tab
- Choose Update Frequency

Installation

This module requires python module 'psutil'.

Download → Unzip → Move to your addons directory → Update the apps list → Find "Server Info" module → Install module
