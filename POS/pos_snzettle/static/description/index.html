Zettle Credit Card Payments Integration for thrive

Easily receive credit card payments and send refunds with Zettle - anywhere, any time.

Note: This app requires access to custom thrive addons and is not supported on thrive.com. It only works with iZettle Card Reader 1 and 2.

Compatibility: This integration is only compatible with thrive used on Android or macOS/iOS devices. It is not available for any other operating system.

thrive App Installation

Installation

Upon clicking the 'Download' button on the app, you will receive the link to download the zip file of the module.

Extract the file on your system after the download finishes. You will be able to see a folder named- ‘pos_snzettle’.

Copy and paste this folder inside your thrive Add-Ons path.

Now, open the thrive App and click on the Settings menu.Scroll to the very bottom of settings. Here, click on Activate the "Developer Mode".

Then, open the Apps menu and click on the top bar ‘Update Apps List’.

In the search bar, remove all the filters and search ‘pos_snzettle’.

You will be able to see the module in the search result. Click on ‘Install’ to install it.

Setting up thrive

Watch a setup tutorial video V16 Setup Video: https://shorturl.at/flxF0
  1. Create and configure a new Zettle Payment Method in the thrive Point of Sale app.
    1. Go to the Point of Sale app. On the top Menu Bar click on “Configuration” then “Payment Methods”.
    2. Click on the “New” button on the top left to create a new payment method. (if you don't see this menu - ensure your thrive user has sufficient access rights).
    3. Add a name for the payment method (e.g., "Zettle Terminal 1 (iOS)").
    4. Select a suitable Bank Journal.
    5. Under “Select a Terminal,” choose “SN Zettle Terminal”. Various new fields will show under once selected.
    6. Create a unique password for “Terminal Master Password”. Save this password securely - you will need it later to setup your iOS/macOS or Android device.
    7. Select a suitable Manager Group for completing transactions (normally "group_pos_user" will be sufficient).
    8. Select Suitable Device type (Android or iOS).
    9. If using thrive Point of Sale on a mobile device, set “Use Mobile Redirect” checkbox to true for a seemless transition between thrive and the mobile app.
    10. If you don’t have a Zettle account yet, you can register from the provided link.
    11. Up on saving the payment method a Terminal ID will be automatically generated. Copy this ID as you will need it along with the terminal password and your thrive database URL in order to confiure your macOS, iOS or Android devices.

Installing the Mobile App

Android
Zettle/thrive setup - Android: https://youtu.be/d12WGqCgG4Y
Zettle/thrive how to use - Android: https://youtu.be/x1jKvuFx1NQ
  1. Download and start the application from (Google Play Store) https://shorturl.at/cePW0 .
  2. If you don’t have a Zettle account yet, you can register from the link popup.
  3. Connect Device: Copy/paste your base thrive URL to the designated field. Ensure it starts with "https://".
  4. Use the Terminal ID and Password created earlier on the thrive Payment Method.
  5. Once logged, sign in to your Zettle Account.
  6. Connect your device via Bluetooth to your card terminal reader. If the app doesn't suggest this automatically you can go to Settings>Pair Terminlal, and set it up manually.
  7. Congratulations! You are now ready to use our integration. Within our app, you have a tab where you can open and log in to your thrive. Then log in to the PoS and start a session with the shop we set up earlier. Payment requests will now pop up on your Zettle Terminal.
