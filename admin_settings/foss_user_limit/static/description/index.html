User Limit


This module allows you to set the users limit.Once the user limit is set we cannot create more user than limit.


Highlights


Users limit

This Module is used to control the creation of users,when the maximum number of Users is exceeded.

  • Screenshots

Advantages

  • In this Module, Admin can manually set the users limits.
  • If User Creation Limit exceed, then it generates an error message.
  • It works for Multicompany Architecture.
  • Number of users are configurable.

Go To - > Settings - > Technical -> System Parameters


In value you can set the user limit.


Error popup when user created more than user limit

Yes, the app works perfectly well with thrive Enterprise as well as thrive Community. thrive Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on thrive Online.
