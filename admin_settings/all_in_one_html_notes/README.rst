All In One HTML Notes
=====================
Addon for setup HTML notes in configuration and used this default notes or
customized notes in sales,purchase,invoice and inventory form view and also use
this notes in reports.

Configuration
=============
* Please specify your notes in corresponding model configuration settings

Installation
============
- www.thrive.com/documentation/17.0/setup/install.html
- Install our custom addon
